# Upload Checklist (run through this for EVERY video)

Copy this list before each upload. Consistency is what beats the algorithm,
not tricks.

## Before filming
- [ ] One clear topic the video is about (search it on YouTube — is there demand?)
- [ ] A hook for the first 15 seconds (state the payoff: why keep watching?)
- [ ] Rough outline so you don't ramble (retention drops when you waffle)

## Thumbnail (do this BEFORE you publish)
- [ ] One clear subject, readable on a phone screen
- [ ] Max 3–4 words of text, big and bold
- [ ] High contrast, a face with emotion if it fits your niche
- [ ] Made 2 versions to test (see the A/B tracker, option 3 in the app)

## Title
- [ ] Generated options with the SEO tool (option 2 in the app)
- [ ] Front-loads the keyword (e.g. "Meal Prep for..." not "My thoughts on...")
- [ ] Under ~60 characters so it isn't cut off
- [ ] Creates curiosity without lying (clickbait that under-delivers kills retention)

## Description & tags
- [ ] First 2 lines summarize the value (they show before "...more")
- [ ] Timestamps/chapters added if the video is long
- [ ] Tags pasted from the SEO tool
- [ ] Links to your other relevant video

## After publishing
- [ ] Pin a comment that asks a question (drives engagement)
- [ ] Reply to the first 10–20 comments fast (early engagement helps reach)
- [ ] After 3–7 days, check Analytics > Reach and log CTR in the A/B tracker
